# MMDAutoTrace4 Sh-hei Edition

Modernized version of MMDAutoTrace4 for Google Colab.

## Goal

Produce working VMD files from videos with minimal modifications to the original project.

## Current Target

- Python 3.12
- PyTorch 2.11
- CUDA 12.8
- Google Colab

## Development Policy

- Keep original structure whenever possible.
- Fix only what is necessary.
- One commit = one fix.
- Prioritize a working pipeline over adding new features.
